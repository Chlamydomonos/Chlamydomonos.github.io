Myxobolang是为完成2022年清华大学编译原理课程大作业而设计的语言。以下是Myxobolang的图标：

Myxobolang概述

Myxobolus 是粘孢子虫门的代表属。一种流行的假说认为，粘孢子虫起源于刺胞动物，它们通过一条神奇的演化路径，从多细胞生物退化成了类似单细胞的形态。Myxobolang的名称来源于Myxobolus，代表这门语言从拥有众多关键字的编程语言退化到只有一个关键字。

Myxobolang的特点

正如语言的名称所要表达的那样，Myxobolang最大的特点在于，其只有一个关键字——::。此外，Myxobolang对标识符的约束也非常低——任何连续的非空白字符（除关键字外）均可作为一个标识符。这意为着，在Myxobolang中，用户可以书写一些在其他语言中不可思议的代码——

:: std :: import :: std :: defaults

int main ( ) {
    int 1+1 = 1 + 2 ;
    print ( 1+1 ) ;
}

以上代码将会输出3。
